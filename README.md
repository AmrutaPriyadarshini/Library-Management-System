Library Management System (.NET WebForms)
This is a fully completed and working Library Management System developed using .NET Framework WebForms in C#. The system automates library operations by providing modules for book management, user management, and handling book transactions.

Key Features
User authentication and role-based access (Admin, Librarian, Student)
Add, update, delete, and search books
Issue and return book functionality
Fine calculation for overdue books
Transaction history and reporting
Dashboard with real-time summary
Responsive interface with Bootstrap

Technologies Used
.NET WebForms
C#
SQL Server
HTML
CSS
JavaScript
Bootstrap
ADO.NET for database connectivity

System Roles / Access Flow
On the Home page, all users can view the list of books and search books without logging in.
The Manage Book, Student Management, Issue, and Return modules are accessible only to the Librarian, and require login.
The Contacts page displays the library location and contact information.
